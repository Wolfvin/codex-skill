# Citation Enforcer

## Must Cite
- Function/class/variable behavior
- Config value claims
- Error-source claims
- Dependency/version claims

## Format
- `path/to/file.ext:line`

## Fallback
If citation unavailable:
- verify first, or
- label as `UNVERIFIED` and do not present as fact.
