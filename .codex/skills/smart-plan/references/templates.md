# Smart Plan Templates

## task_plan.md

```markdown
# Plan: [Task Name]
Created: [YYYY-MM-DD HH:MM]
Status: IN_PROGRESS

## Goal
[Outcome when task is done]

## Success Criteria
- [ ] [Criterion 1]
- [ ] [Criterion 2]

## Phases
### Phase 1: [Name]
- [ ] TASK: [Task] -> verify: [Verification]

### Phase 2: [Name]
- [ ] TASK: [Task] -> verify: [Verification]

## Blockers and Decisions
| Issue | Decision | Reason |
|-------|----------|--------|
```

## findings.md

```markdown
# Findings: [Task Name]

## Technical Decisions
- [Decision] -> [Reason]

## Research Notes
- [Key findings]

## Known Gotchas
- [Risk and mitigation]
```

## progress.md

```markdown
# Progress Log: [Task Name]

## Session [YYYY-MM-DD HH:MM]
- Start: [Context]
- Completed: [Result]
- Next: [Next action]

## Error Log
| Error | Fix | Timestamp |
|-------|-----|-----------|

## Test Results
| Test | Status | Notes |
|------|--------|-------|
```
