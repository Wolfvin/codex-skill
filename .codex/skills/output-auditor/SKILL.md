# Output Auditor

## 4-Angle Check
1. Factual accuracy
2. Logical consistency
3. Completeness against request
4. Safety/regression risk

## Severity
- CRITICAL: block delivery
- WARNING: deliver with caveat
- MINOR: optional polish
