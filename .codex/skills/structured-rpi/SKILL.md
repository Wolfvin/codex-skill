# Structured RPI (Research-Plan-Implement)

## Phases
1. Questions
2. Research
3. Design Discussion
4. Structure Outline
5. Plan (with acceptance criteria)
6. Implement + verify

## Gate Rule
Stop after each phase and request approval before advancing.

## Output
- A short markdown artifact per phase.
- Clear gate prompt: approve / request changes / skip / abort.
